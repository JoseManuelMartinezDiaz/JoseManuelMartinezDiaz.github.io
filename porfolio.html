Beer'em Up (Team Project)

This is a game i developed with a team of 15 people forming a team inside the university. Beer'em Up is a combat game based on the genre Beat'em up where the main characters: Duffer and Ambar must confront different enemies and obstacles to defeat Raisin, the nemensis of our heroes.

The game was made in Unreal Engine 4 with C++ and published in Steam.

Features

  • Main characters' mechanics design
  • Implementation of a particle pooling system to improve performance during the profiling phase of the project
  • System based on the UE4´s destructible objects.
  • Design and implementation of gameplay elements such as: consumible pickups or chests.
  • Creation and implementation of VFX elements

Destructible objects system in Unity (Personal Project)

This tool encapsulate the functionalities of a C# wrapper of Nvidia Blast C++ library that uses Voronoi to create destructible objects.

One of the main priorities of this library was to be stable and do not create a performance drop on the game or product. Usually is it hard to find tools that achieve destruction without been slow in performance or even buggy. This tool may not be the most realistic approach but in terms of performance it success compared to other products. This is thanks to the implementation of Nvidia Blast that optimizes in a tremendous way the mathematical algorithms and in the same situation achieves in a few seconds what other tools cost around 30-60 seconds.

Features

  • C#
  • Unity 3D
  • Voronoi and quick hull algorithm
  • Unity's fixed joint system
  • NVidia Blast
